## 🔧 Troubleshooting

### If Vercel Build Fails
1. Check `npm run build` locally - if it works locally, Vercel will work
2. Verify `VITE_API_BASE_URL` is set correctly in Vercel dashboard
3. Check Vercel build logs for specific TypeScript or npm errors

### If Railway Deployment Fails
1. Check `python manage.py check` locally - if it passes, Railway will pass
2. Verify all environment variables are set in Railway dashboard
3. Ensure PostgreSQL plugin is added to the Railway project
4. Check Railway deployment logs for migration or gunicorn errors

### CORS Issues Between Frontend & Backend
1. Verify `CORS_ALLOWED_ORIGINS` in Railway exactly matches Vercel URL
2. No trailing slashes: `https://app.vercel.app` (not `https://app.vercel.app/`)
3. Both services must be using HTTPS (not HTTP)

### Static Files Not Loading on Vercel
1. This shouldn't happen - Vercel serves the `dist/` folder directly
2. If issues, check that Vite build outputs to `dist/` folder
3. Verify `vercel.json` routes configuration
